# /Skills — องค์ความรู้ที่ใช้ซ้ำได้

เก็บ **Skill** = how-to / gotcha ที่เรียนรู้จากงานจริงและนำกลับมาใช้ซ้ำได้

## รูปแบบไฟล์
หนึ่ง Skill = หนึ่งไฟล์ `.md` มี frontmatter (ดู [_template.md](_template.md)):
```yaml
---
name: <kebab-case-name>
description: <หนึ่งบรรทัด: ทำอะไร / ใช้เมื่อไร>
domain: backend | frontend | db | general
updated: YYYY-MM-DD
---
```

## กฎเหล็ก (สำคัญ)
- **อัปเดตเมื่อมีความรู้ใหม่** — เจอ gotcha/วิธีทำใหม่ ให้บันทึกทันที
- **ห้ามสร้างข้อมูลซ้ำ** — ถ้ามีหัวข้อใกล้เคียงอยู่แล้วให้แก้ไฟล์เดิม + อัปเดต `updated:` ไม่สร้างไฟล์ใหม่ซ้อน
- ก่อนเริ่มงานให้สแกนโฟลเดอร์นี้ตาม `domain` ของงาน (ดู [Hooks/before-task.md](../Hooks/before-task.md))

## ดัชนี (Index)
